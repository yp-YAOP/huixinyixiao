🚀 慧心康复训练中心 - 服务器启动帮助

📝 说明：由于需要本地服务器来运行游戏，请按照以下任一方法启动HTTP服务器。

方法1: Python (推荐)

如果您的系统安装了Python：

python -m http.server 8080

或者对于Python 2：

python -m SimpleHTTPServer 8080

方法2: Node.js

如果您的系统安装了Node.js：

npx http-server -p 8080

或者先安装后使用：

npm install -g http-server
http-server -p 8080

方法3: PHP

如果您的系统安装了PHP：

php -S localhost:8080

方法4: Live Server (VS Code)

如果您使用VS Code：

  1. 安装 "Live Server" 扩展
  2. 右键点击任意HTML文件
  3. 选择 "Open with Live Server"
✅ 启动成功后，您将看到：

服务器运行在 http://localhost:8080 或类似地址

🔧 故障排除

如果遇到问题：

  1. 确保没有其他程序占用8080端口
  2. 尝试使用不同的端口号（如8000、3000等）
  3. 检查防火墙设置
  4. 使用Chrome或Edge浏览器（推荐）
